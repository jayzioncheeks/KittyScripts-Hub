Introduction

XENO is a refined UI library for Roblox scripts, optimized for performance and production use. It gives you a complete set of UI primitives like windows, tabs, elements, search bars, and notifications.

lua 
local XENO = loadstring(game:HttpGet(('https://raw.githubusercontent.com/zxkuhl/XENO/refs/heads/main/XENO-PUBLIC')))()
Overview › Getting Started

Getting Started

Initialization #

Include the library and wait for the game environment to initialize. The wait(6) ensures everything loads smoothly.

lua 
local XENO = loadstring(game:HttpGet(('https://raw.githubusercontent.com/zxkuhl/XENO/refs/heads/main/XENO-PUBLIC')))()
wait(6)
Configuration #

Before building your UI, you can easily set your script's title and link a community Discord server.

lua 
--- Configuration ---
_G.dc = "https://discord.gg/yourdc"
_G.XENOGUINAME = "name"
Theming #

XENO allows you to adjust the visual style dynamically. You can enforce a default theme using SetTheme(), or provide users with buttons to change it themselves.

lua 
SetTheme("Purple")

AddThemeBtn("Purple", Themes.Purple.Accent1)
AddThemeBtn("Green",  Themes.Green.Accent1)
AddThemeBtn("Blue",   Themes.Blue.Accent1)
AddThemeBtn("Red",    Themes.Red.Accent1)

Elements & Commands

XENO includes several components to build out your interface. The final boolean argument usually determines if a notification is triggered.

Status Bar & Search #

Adds a search bar to the top of your UI to filter through features easily.

lua 
XENO:AddSearchBar()
XENO:AddHeader("NORMAL FEATURES")
XENO:AddLabel("Standard interactions without alerts")
Dropdowns #

Standard dropdowns allow single selection, while Multi-Select dropdowns return a table of chosen strings.

lua 
XENO:AddDropdown("Teleport List", {"Spawn", "Bank", "Safe Zone"}, "Spawn", function(opt)
    print("Teleporting to:", opt)
end, false)

XENO:AddMultiSelectDropdown("Select Zones", {"Spawn", "Bank", "Arena"}, {"Spawn"}, function(selected)
    print("Selected:", table.concat(selected, ", "))
end, false)
Interactive Elements #
lua 
XENO:AddSwitch("Auto-Farm", function(state) end, true)
XENO:AddSlider("WalkSpeed", 16, 100, 16, function(val) end)

Full Implementation

This script includes Configuration, ToggleUI logic, and Player Variable handling to ensure it works in-game immediately.

lua 
--- Configuration ---
_G.dc = "https://discord.gg/yourdc"
_G.XENOGUINAME = "name"

local XENO = loadstring(game:HttpGet(('https://raw.githubusercontent.com/zxkuhl/XENO/refs/heads/main/XENO-PUBLIC')))()
local Players = game:GetService("Players")
local LocalPlayer = Players.LocalPlayer

wait(6)

XENO:AddSearchBar()

local isVisible = true
local function ToggleUI()
    isVisible = not isVisible
    XENO:SetVisible(isVisible)
end

XENO:AddHeader("NORMAL FEATURES")

XENO:AddDropdown("Teleport List", {"Spawn", "Bank", "Safe Zone"}, "Spawn", function(opt)
    print("Teleporting to:", opt)
end, false)

XENO:AddMultiSelectDropdown("Select Modes", {"Legit", "Blatant", "Rage", "Silent"}, {"Legit"}, function(selected)
    print("Modes selected:", table.concat(selected, ", "))
end, true)

XENO:AddSlider("WalkSpeed", 16, 100, 16, function(val)
    if LocalPlayer.Character and LocalPlayer.Character:FindFirstChild("Humanoid") then
        LocalPlayer.Character.Humanoid.WalkSpeed = val
    end
end)

XENO:AddHeader("SETTINGS & THEMES")

AddThemeBtn("Purple", Themes.Purple.Accent1)
AddThemeBtn("Green",  Themes.Green.Accent1)
AddThemeBtn("Red",    Themes.Red.Accent1)

XENO:AddKeybind("Toggle Interface", Enum.KeyCode.RightControl, function()
    ToggleUI()
end, true)
